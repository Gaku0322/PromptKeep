# Prompt Keep

AIチャットやライティングツールで頻繁に使うプロンプトを、効率的に管理・保存するためのChrome拡張機能です。

## 特徴
- 完全ローカル管理: データはブラウザのストレージにのみ保存されます。
- タグ検索: 増え続けるプロンプトをタグで素早くフィルタリング。
- ワンクリックコピー: 必要な時にすぐクリップボードへ。
- 自動ダークモード: OSの設定に合わせて快適な配色に切り替わります。

## 導入方法
1. 本リポジトリをZIPでダウンロードまたは `git clone` します。
2. Chromeの拡張機能管理画面 (`chrome://extensions/`) を開きます。
3. 「デベロッパー モード」をONにします。
4. 「展開して読み込み」をクリックし、本フォルダを選択します。
